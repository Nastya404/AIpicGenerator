# AI Image Generator

Веб-приложение для генерации изображений с помощью искусственного интеллекта.

## Возможности

- Генерация изображений по текстовому описанию
- Поддержка нескольких моделей ИИ (FLUX, Stable Diffusion, Openjourney)
- Темная/светлая тема
- Скачивание сгенерированных изображений
- Генерация случайных промптов
- Адаптивный дизайн

## Используемые технологии

- HTML5
- CSS3
- JavaScript (Vanilla)
- [Pollinations.ai API](https://pollinations.ai) - бесплатный API для генерации изображений

## Доступные модели

- **FLUX.1-dev** - Высокое качество, детализированные изображения
- **FLUX.1-schnell** - Быстрая генерация с хорошим качеством
- **Stable Diffusion XL** - Универсальная модель
- **Stable Diffusion v1.5** - Классическая быстрая модель
- **Openjourney** - Художественные стили

## Как использовать

1. Откройте `index.htm` в браузере
2. Введите описание изображения в текстовое поле
3. Выберите модель, количество изображений и соотношение сторон
4. Нажмите "Generate"
5. После генерации наведите курсор на изображение и нажмите кнопку загрузки

## Особенности

- При выборе нескольких изображений каждая картинка будет уникальной
- Поддержка различных соотношений сторон (1:1, 16:9, 9:16)
- Сохранение выбранной темы в браузере
- Обработка ошибок и таймаутов
